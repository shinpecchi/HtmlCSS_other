## 授業で説明した順番<br>
### ①アイコンをあと２つ追加＆styleに色を設定<br>
```
<div class="col-sm-4">
	<i class="bi-lightbulb" style="font-size: 4rem;color: blue;"></i>
</div>
<div class="col-sm-4">
	<i class="bi-eyedropper" style="font-size: 4rem;color: red;"></i>
</div>
<div class="col-sm-4">
	<i class="bi-globe" style="font-size: 4rem;color: darkgreen;"></i>
</div>
```
<br>

### ②divタグのクラスに `text-center` を追加

```
<div class="col-sm-4 text-center">
	<i class="bi-lightbulb" style="font-size: 4rem;color: blue;"></i>
</div>
<div class="col-sm-4 text-center">
	<i class="bi-eyedropper" style="font-size: 4rem;color: red;"></i>
</div>
<div class="col-sm-4 text-center">
	<i class="bi-globe" style="font-size: 4rem;color: darkgreen;"></i>
</div>
```
<br>

### ③アイコンタグの前にpタグ `<p></p>` を追加

```
<div class="col-sm-4 text-center">
	<p><i class="bi-lightbulb" style="font-size: 4rem;color: blue;"></i></p>
</div>
<div class="col-sm-4 text-center">
	<p><i class="bi-eyedropper" style="font-size: 4rem;color: red;"></i></p>
</div>
<div class="col-sm-4 text-center">
	<p><i class="bi-globe" style="font-size: 4rem;color: darkgreen;"></i></p>
</div>
```
<br>

### ④アイコンタグの下にpタグで文字を追加
追加する文字<br>
```
<p>あいいうえおあいうえおあいうえおあいうえおあいいうえおあいうえおあいうえおあいうえお<br>
かきくけこ</p>
```
<br>

### ⑤CSSで `h2` の下に追加
```
.fadeUp{
	animation-name:fadeUpAnime;
	animation-duration:3s;
	animation-fill-mode:forwards;
	opacity:0;
}
```
<br>

### ⑥CSSで `.fadeUp` の下に追加
```
@keyframes fadeUpAnime{
	from{
		opacity: 0;
		transform: translateY(50px);
	}
	
	to{
		opacity: 1;
		transform: translateY(0);
	}
}
```
